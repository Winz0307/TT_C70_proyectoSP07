# TT_C70_proyectoSP07
Dashboard interactivo del proyecto SP07
Se realiza una dashboard interactivo con un encabezado que indica "Analisis de Ventas de vehiculos usados"
se observan dos botones, uno que realiza un histograma para el conjunto de datos de anuncios de venta de coches. Y un segundo boton tipo checkbox que permite al usuario desplegar y remover un grafico de dispersion a gusto.
